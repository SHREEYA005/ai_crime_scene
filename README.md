# AI-Based Crime Scene Reconstruction

A full-stack AI-powered forensic platform for crime scene analysis and 3D reconstruction.

## Team
- **You** — Python / Backend (Flask, YOLOv8, OpenCV, Database, PDF)
- **Her** — Unity / Frontend (3D scene, VR, annotation canvas, HTML templates)

## Quick Start
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
pip install -r requirements.txt
python run.py
```
Then open: http://localhost:5000/auth/login
